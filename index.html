Características

Principais características

Projeto open source, atuando como guia de estilo de desenvolvimento de código na camada de negócios.

Guias de Estilo

Praticando o princípio aberto-fechado, ajudamos os desenvolvedores a construir casos de uso uniformes de forma nativa.

Multilinguagem

Não exigindo novos aprendizados, ferramentas ou linguagens. Atuamos de forma nativa ao desenvolvedor, focando na aprendizagem de semanticas em uma sintaxe já conhecida.

class GETUseCase : UseCase {
  var repo: UseCaseRepository
  
  init(repo: UseCaseRepository) {
      self.repo = repo
  }
  
  override func execute(param: String?) -> Output {
      let fetch = repo.doFetch(name: param)
      return ValueOutput(value: fetch)
  }
}
                  
                
                  
class GETAPIUseCase(UseCase[int, HousesToReturn]):
def __init__(self, repo: RepositoryAPI):
    self.repo = repo

def execute(self, param: int) -> Output[HousesToReturn]:
    print("GETAPIUseCase.execute")
    site = self.repo.do_fetch(page=param)
    houses = self.repo.read_tags_from_site(site)
    
    return ValueOutput(HousesToReturn(houses))

{

        this.repo = repo;
    }

    public override Output execute(string param)
    {

    }
